# Security Policy

## Scope

This project is a local development environment setup tool. It installs
and configures CLI tools on your own machine. It does not collect, store,
or transmit user data, and it does not expose any network services.

## Supported Versions

Only the latest commit on the `main` branch is supported.

## Reporting a Vulnerability

If you discover a security issue, please report it through
[GitHub's private vulnerability reporting][pvr] on this repository.

Do **not** open a public issue for security vulnerabilities.

[pvr]: https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-creating-advisories/privately-reporting-a-security-vulnerability
